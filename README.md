# Je Yun Kim's Portfolio

A modern, responsive portfolio website built with React.

## Overview
This portfolio showcases my work experience, skills, and current projects. Visit the live site at [https://jeyunkim.github.io/webpage](https://jeyunkim.github.io/webpage)

## Technologies Used
- React
- Styled Components
- React Router
- Font Awesome

## Local Development
1. Install dependencies: `npm install`
2. Run development server: `npm start`
3. Build for production: `npm run build`
4. Deploy to GitHub Pages: `npm run deploy`
